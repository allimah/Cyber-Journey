# الأسبوع الثالث: أساسيات Linux - الجزء الأول

هذا الأسبوع يركز على تعلم الأوامر الأساسية للتفاعل مع نظام التشغيل Linux من خلال سطر الأوامر (Terminal).

## 1. أوامر التنقل والإنشاء

هذه هي الأوامر التي تسمح لنا بالتحرك داخل نظام الملفات وإنشاء بنية أساسية للمشاريع.

| الأمر | الوظيفة | مثال للاستخدام |
| :--- | :--- | :--- |
| `mkdir` | **Make Directory** (أنشئ مجلدًا): يقوم بإنشاء مجلد جديد بالاسم المحدد. | `mkdir My-Project` |
| `cd` | **Change Directory** (غيّر المجلد): يقوم بنقلك إلى المجلد المحدد. | `cd My-Project` |
| `touch` | **Touch** (المس): يقوم بإنشاء ملف جديد فارغ إذا لم يكن موجودًا. | `touch notes.txt` |
| `rm` | **Remove** (احذف): يقوم بحذف الملفات المحددة. (يجب استخدامه بحذر). | `rm notes.txt` |
| `cd ..` | **Change Directory to Parent** (غيّر إلى المجلد الأب): يقوم بنقلك مستوى واحدًا للأعلى في شجرة المجلدات. | `cd ..` |

### ملاحظة هامة: دليل العمل الحالي (Current Working Directory)

الأوامر التي يتم تنفيذها تؤثر فقط على الموقع (المجلد) الذي نقف فيه حاليًا. يمكننا دائمًا معرفة موقعنا الحالي باستخدام الأمر `pwd`. محاولة حذف ملف غير موجود في دليل العمل الحالي ستؤدي إلى خطأ، وهذا سلوك وقائي مهم.


## 2. أوامر إدارة الملفات

هذه الأوامر تسمح لنا بالتعامل مع محتويات الملفات ونسخها ونقلها.

| الأمر | الوظيفة | مثال للاستخدام |
| :--- | :--- | :--- |
| `nano` | **Nano Editor**: محرر نصوص بسيط يعمل داخل الـ Terminal لإنشاء وتعديل الملفات النصية. | `nano myfile.txt` |
| `cat` | **Concatenate**: يعرض محتويات ملف نصي على الشاشة. | `cat myfile.txt` |
| `cp` | **Copy**: ينسخ الملفات أو المجلدات من مصدر إلى وجهة. | `cp source.txt destination.txt` |
| `mv` | **Move**: ينقل أو يعيد تسمية الملفات أو المجلدات. | `mv old_name.txt new_name.txt` |
